Sample graphviz chapter

General information

The documentation to graphviz and thus an explanation how to generate this graphs is available under: https://www.graphviz.org/pdf/dotguide.pdf

Further, an free online editor can be accessed via: https://dreampuf.github.io/GraphvizOnline

An idea could be to write a short python script which can be called in order to automatically generate this graphs. Otherwise, they need to be generated in the “sphinx” folder just via the shell command “make html”.

Attention

Besides “pip install graphviz” to the desired environment, which may causes some issues, it is recommend to install it via “sudo apt-get install graphviz”.
